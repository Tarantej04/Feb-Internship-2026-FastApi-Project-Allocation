# 🍔 QuickBite Food Delivery API

A FastAPI-based backend project for a food delivery system. This API allows users to browse menu items, manage cart, place orders, and perform advanced operations like filtering, sorting, searching, and pagination.

---

## 🚀 Features

- 📋 View Menu Items
- 🔍 Search Menu
- 🔽 Sort Menu
- 📄 Pagination
- 🧠 Smart Browse (Search + Sort + Pagination)
- 🛒 Cart Management (Add, Remove, View, Checkout)
- 📦 Order Management
- ✏️ Update Menu Items
- ❌ Delete Menu Items
- ➕ Add New Menu Items (with validation)

---

## 🛠️ Tech Stack

- Python 3
- FastAPI
- Uvicorn
